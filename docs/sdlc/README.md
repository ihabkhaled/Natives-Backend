# SDLC Baselines

These documents are the permanent company baselines. They define the standing rules that every request-specific feature folder must inherit and follow.

They should be read as operating manuals, not short summaries. If a new recurring expectation appears in delivery, testing, release, security, QA, documentation, or support, it belongs in this folder and in `claude.md`.
